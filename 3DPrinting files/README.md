## English

The 3D files in this folder are modifications and add-ons for Martin's official MakerWorld project.

You can find the original project here:

- https://makerworld.com/de/models/2380835-full-size-modular-vending-machine#profileId-2607127

The MakerWorld-hosted modifications are available here:

- https://makerworld.com/de/models/2678974-sumup-solo-terminal-mod-esp23-new-housing#profileId-2966622

Another MakerWorld project for the motor ESP modifications and the parts mounted underneath the machine is available here:

- https://makerworld.com/de/models/2624323-esp32-keypad-housing-backplate-modification#profileId-2897053

Please note that these files do not replace the original base project. They are intended as custom additional or modified parts for it. The modifications in this repository and the MakerWorld versions are maintained in parallel. The files from the motor ESP and bottom-side project are also included in this GitHub repository.
